Cyber Essentials questions explained (plain English)

Most SMEs fail Cyber Essentials on interpretation and evidence, not intent. Here’s how to think about the self-assessment questions so you can answer confidently — and build an evidence pack your MSP/assessor can work with.

cyber essentials questionnaire self-assessment evidence

What the questions are really testing

  • Consistency: is the control applied everywhere (not “one good laptop”)?
  • Ownership: does someone clearly own the control and its maintenance?
  • Proof: can you produce evidence quickly without a scavenger hunt?

How to answer without overthinking

Write the minimum true answer

Short, factual, and consistent with your evidence. If a control is “in progress”, document the plan and owner — don’t pretend it’s complete.

Attach evidence per answer

For each answer, decide what evidence would convince an assessor: a screenshot, a policy note, an export, or a statement from your MSP.

Evidence prompts that cover most questions

  • MFA enforced for all users + admins (screenshots/policy)
  • Device encryption enabled (BitLocker/FileVault evidence)
  • AV/Defender enabled and protected from tampering
  • Patching policy + proof updates are enabled
  • User list + admin roles + leavers process note
  • Firewall/router ownership + configuration proof (often via MSP)

If you can produce the above quickly, tenders stop feeling like last-minute firefights.
