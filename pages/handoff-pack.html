Sample handoff pack (outline)

The goal is a single export that makes it obvious what’s done, what’s missing, and where the evidence lives — so your MSP/assessor can move quickly.

gap summary evidence index control notes

What’s inside

  1. Cover: company, prepared-by, export date, target standard/version
  2. Executive readiness summary (1 page): status (Ready / At risk / Not ready), score, top blockers, next 7 days plan
  3. Environment snapshot: M365/Google, device counts, remote access methods, in-scope services
  4. Control-by-control status: owner, notes, evidence list, open questions, recommended remediation
  5. Evidence index: a table listing each evidence item, what it proves, where it lives, and which control it maps to
  6. Missing evidence checklist: expected items per control marked present/missing
  7. MSP/assessor questions list: auto-generated from gaps (optional)
Why this matters: Cyber Essentials work often stalls because evidence is scattered. The pack makes the handoff clean.
